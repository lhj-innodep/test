설치 사양 확인하기

원활한 앱 실행을 위해 설치 사양을 확인합니다.

지원 OS

•Andriod 10 이상 지원

•iOS / iPad 14.0 이상 지원

지원 코덱

•H.264 지원

•H.265 지원

권장 VURIX 서버 버전

•1.7.0.220214 이상 권장

권장 웹 API 버전

•1.7.0.220214 이상 권장

참 고

•VURIX 모바일은 웹 API를 통해 연결되므로 모바일 네트워크 환경에서 웹 API 서버의 IP가 외부에서 접근 가능해야 합니다.

•웹 API 서버가 외부 접근이 가능하도록 설정되어 있다면 웹 API 서버 설정에서 RTSP 서비스 IP 및 포트 포워딩 설정이 정확하게 적용되었는지 반드시 확인해야 합니다.

앱 설치하기

모바일 앱을 설치합니다.

❶

❷

❸

1.[Play 스토어] 또는 [App Store]에서 "VURIX"를 검색하세요.

2.[설치]를 누르세요.

3.앱을 실행하세요.

참 고

•iOS 기기는 로컬 네트워크 [허용 안 함] 설정이 필요합니다.

•iOS 기기에서 [설정] > [앱] > [VURIX] > [로컬 네트워크]를 누르세요. 로컬 네트워크 설정하세요.

기본 작동하기

로그인 및 로그아웃

실제 화면과 설명서 이미지는 다를 수 있습니다.

로그인하기

로그인 정보를 입력해 앱에 접속합니다.

❶

❷

1.로그인 정보를 입력하세요.

2.[로그인]을 누르세요. 앱에 접속합니다.

참 고

앱에 접속한 계정이 로그인 목록에 저장됩니다.

다른 계정으로 로그인하기

다른 계정으로 앱에 접속합니다.

❶

❷

❸

1.로그인 화면에서 [다른 계정 사용하기]를 누르세요.

2.로그인 정보를 입력하세요.

3.[로그인]을 누르세요. 앱에 접속합니다.

참 고

•앱에 접속한 계정이 로그인 목록에 저장됩니다.

•[계정 삭제하기]를 누르세요. 로그인 목록에 저장된 계정을 삭제합니다.

로그아웃하기

앱에서 로그아웃합니다.

❶

❷

❸

1.메인 화면에서 []를 누르세요. 마이 페이지 화면을 표시합니다.

2.[로그아웃]을 누르세요.

3.[완료]를 누르세요. 앱에서 로그아웃합니다.

실시간 영상 모니터링하기

실제 화면과 설명서 이미지는 다를 수 있습니다.

라이브

실시간 영상을 확인합니다.

❶

❷

1.메인 화면에서 []를 누르세요. 실시간 영상 화면을 표시합니다.

2.화면을 왼쪽, 오른쪽으로 미세요.

실시간 영상을 확인합니다.

참 고

•실시간 영상은 최대 16 채널까지 할당 가능합니다.

•화면에 동시 표출 가능한 영상은 최대 4 채널입니다.
